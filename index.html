Welcome to ACNH Checklist

This site helps you track collection progress in Animal Crossing: New Horizons (ACNH). Use the checklists to track K.K. Slider songs, fish, bugs, fossils, and other collectibles.

K.K. Songs Checklist Test Page

How to use this checklist

  • Click on a checklist (e.g., K.K. Songs) to view items.
  • Use the search to quickly find items.
  • Filter status (collected / not collected) with the dropdown.
  • Collected states persist using your browser's localStorage — nothing is uploaded.

Privacy & Data

This site stores only local data in your browser (cookies/localStorage) — nothing is sent to a server.
